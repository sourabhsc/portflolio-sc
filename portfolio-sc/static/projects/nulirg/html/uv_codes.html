Extended description of function.

  • gal_num (int) – galaxy index (0,1,2,3,4)
  • config_file (str) – the configuration file that contains most of the parameters required for ULIRG codes (e.g. ULIRG_params.conf)
  • section (str) – the section of config file with basic parameters for the runs (e.g. ‘basic’)
